## Pre-requisites

You need Python >=3.9 on your machine to install `csp-bot`.

## Install with `pip`

```bash
pip install csp-bot
```

## Install with `conda`

```bash
conda install csp-bot --channel conda-forge
```

## Source installation

For other platforms and for development installations, [build `csp-bot` from source](Build-from-Source).
